About the Randomizer:

This randomizer currently randomizes Chests and Stickers across all three playable characters. More functionality will be added over time as this randomizer gets developed.

How to get started:

You need LuaFrontend or LuaBackend for the randomizer to work. For LuaFrontend it works just like the other games (A guide can be find on the respective download on github). If you're using LuaBackend make sure to create a folder named "bbs" in your scripts folder of the game, then place the seed.lua file in there.
